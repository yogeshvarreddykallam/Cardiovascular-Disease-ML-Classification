# Cardiovascular Disease ML Classification

Replication and extension of a machine learning fusion approach for predicting cardiovascular disease severity. Implements binary and multi-class classification pipelines across multiple models.

## Notebooks
| Notebook | Task |
|---|---|
| `binary_classification.ipynb` | Presence vs absence of cardiovascular disease |
| `multiclass_classification.ipynb` | Severity level prediction (mild / moderate / severe) |

## Models Compared
Logistic Regression, Random Forest, Gradient Boosting, SVM, and fusion ensemble

## Files
| File | Description |
|---|---|
| `binary_classification.ipynb` | Binary classification pipeline |
| `multiclass_classification.ipynb` | Multi-class severity classification |
| `replication_report.pdf` | Full replication study report |
| `presentation.pptx` | Project presentation |
